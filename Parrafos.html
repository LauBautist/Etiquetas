¿Qué es un párrafo?

En HTML, un párrafo se define utilizando la etiqueta <p>. Un párrafo es un bloque de texto que siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco antes y después de él.

Los párrafos se utilizan para agrupar oraciones y textos relacionados en un documento HTML, proporcionando una estructura clara y legible.

Ejemplo básico de un párrafo en HTML: 

 <p>Este es un párrafo en HTML.</p>

Puedes agregar tantos párrafos como necesites dentro de una página HTML.

Reglas Horizontales HTML

La etiqueta <hr> define un salto temático en una página HTML y generalmente se muestra como una regla horizontal.

El elemento <hr> se utiliza para separar contenido (o definir un cambio) en una página HTML.

Saltos de línea HTML

El Elemento HTML <br>

El elemento HTML <br> define un salto de línea. Es útil para crear saltos de línea dentro de un párrafo sin empezar un nuevo párrafo. 

            <p>Ejemplo de salto de línea:<br>
            <p>Este es el primer texto.<br>
            Y este es el segundo, después del salto de línea.</p>

Ejemplo de salto de línea:
Este es el primer texto.
Y este es el segundo, después del salto de línea.

El Elemento HTML <pre>

El elemento HTML <pre> define texto preformateado. El texto dentro de un <pre> elemento se muestra en una fuente de ancho fijo (generalmente Courier) y conserva tanto los espacios como los saltos de línea. 

    Ejemplo de texto
    con varios espacios
    y saltos de línea preformateados.

Referencia de Etiquetas HTML

Tag Description
<p> Define un párrafo
<hr> Define un cambio temático en el contenido
<br> Inserta un salto de línea
<pre> Define texto preformateado
