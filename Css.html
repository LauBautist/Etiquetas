1. Fundamentos de CSS, Modelo de Caja y Layout

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir el estilo y el diseño de una página web. Funciona junto con HTML para definir cómo se verá el contenido en una página.

Sintaxis de CSS

La sintaxis básica de CSS incluye:

selector {
    propiedad: valor;
}

Los selectores especifican qué elementos HTML se van a estilizar, mientras que las propiedades y valores definen el estilo.

Modelo de Caja

El Modelo de Caja es fundamental para entender cómo los elementos HTML son renderizados. Cada elemento se representa como una caja que consta de márgenes, bordes, padding y el contenido.

elemento {
    margin: 10px;
    padding: 20px;
    border: 2px solid #333;
}

Diseño de Layout (Flexbox y Grid)

Existen varias técnicas en CSS para crear diseños de páginas web responsivos. Dos de las más populares son Flexbox y Grid Layout.

Ejemplo con Flexbox

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Ejemplo con Grid Layout

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

2. Texto, Colores y Multimedia

Estilización de Texto

CSS proporciona muchas propiedades para estilizar texto, como tamaño de fuente, color y alineación.

p {
    font-size: 16px;
    color: #333;
    text-align: center;
}

Colores y Fondos

En CSS puedes trabajar con colores en diferentes formatos: HEX, RGB, HSL, entre otros.

body {
    background-color: #f0f0f0;
    color: #333;
}

Imágenes y Multimedia

Con CSS puedes ajustar el tamaño, bordes y sombras de las imágenes.

img {
    width: 100%;
    border: 2px solid #333;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

También es posible establecer imágenes de fondo:

body {
    background-image: url('imagen.jpg');
    background-size: cover;
}

3. Transiciones, Animaciones y Estilos Avanzados

Transiciones

Las transiciones suavizan los cambios de estilo, permitiendo efectos animados.

elemento {
    transition: all 0.3s ease;
}

Animaciones

Las animaciones permiten cambios más complejos.

@keyframes movimiento {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}

elemento {
    animation: movimiento 2s infinite;
}

Estilos Avanzados

CSS moderno permite el uso de pseudo-elementos, variables CSS y funciones como :nth-child().

elemento::before {
    content: '→';
    color: red;
}

:root {
    --primary-color: #4CAF50;
}

elemento {
    color: var(--primary-color);
}

4. Frameworks, Accesibilidad y Optimización

Frameworks CSS

Frameworks como Bootstrap o Tailwind facilitan el desarrollo web al proporcionar componentes predefinidos.

Accesibilidad en CSS

La accesibilidad es crucial para garantizar que todos los usuarios, incluidas personas con discapacidades, puedan navegar por una página web.

elemento {
    font-size: 1.2em;
    color: #333;
}

Optimización y Herramientas de Desarrollo

Minificar y agrupar archivos CSS puede mejorar la velocidad de carga. Herramientas como Chrome DevTools permiten depurar CSS.

/* Ejemplo de CSS minificado */
body{background:#fff;color:#333;margin:0;}
