¿Qué son los formatos en HTML?

Los formatos en HTML son etiquetas que permiten modificar el aspecto de un texto en una página web, como por ejemplo, ponerlo en negrita, cursiva, subrayado, tacharlo, o cambiar su tamaño. Estas etiquetas se utilizan para resaltar información, mejorar la lectura o llamar la atención del usuario.

Ejemplo:

        <!DOCTYPE html>
        <html lang="es">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Estilos básicos HTML</title>
        </head>

        <body>
            <p><b>Este texto esta en negrita</b></p>
            <p><i>Este texto esta en cursiva</i></p>
            <p>Esto es un<sub>índice</sub> y <sup>super índice</sup></p>
        </body>
        </html>

Elementos de formato HTML

Los elementos de formato fueron diseñados para mostrar tipos especiales de texto:

  • <b> - Texto en negrita
  • <strong> - Texto importante
  • <i> - Texto en cursiva
  • <em> - Texto destacado
  • <mark> - Texto marcado
  • <small> - Texto más pequeño
  • <del> - Texto eliminado
  • <ins> - Texto insertado
  • <sub> - Texto en subíndice
  • <sup> - Texto en superíndice

Ejemplos de cada elemento

Elementos HTML <b> y <strong>:

  • <b> y <strong>: El elemento HTML <b> define texto en negrita, sin ninguna importancia extra
 <b>Este es un texto en negrita</b>

En este ejemplo, el texto "Este es un texto en negrita" aparece en negrita, sin que el navegador le asigne ninguna importancia adicional.

  • <strong>: El elemento HTML <strong> define texto con gran importancia. El contenido que se encuentra dentro de él normalmente se muestra en negrita.
 <strong>Este en un texto importante</strong>

En este ejemplo, el texto "Este es un texto importante" aparece en negrita, pero con una implicación semántica de importancia.

Elementos HTML <i> y <em>:

  • <i> y <em>: El elemento HTML <i> define una parte del texto en una voz o estilo alternativo. El contenido dentro de él normalmente se muestra en cursiva.
 <i>Este es un texto en cursiva</i>

En este ejemplo, el texto "This text is italic" aparece en cursiva, indicando una voz o estilo alternativo.

  • <em>: El elemento HTML <em> define texto resaltado. El contenido que se encuentra dentro de él normalmente se muestra en cursiva.
    •  <em>Este texto esta resaltado</em>

    En este ejemplo, el texto "Este texto esta resaltado" aparece en cursiva, indicando un énfasis especial, que un lector de pantalla podría pronunciar con énfasis adicional.

    Elementos HTML <small>:

    • <small>: El elemento HTML <small> define un texto más pequeño. 
     <small>Este es un texto más pequeño</small>

    En este ejemplo, el texto "Este es un texto más pequeño" aparece más pequeño que el texto normal.

    Elementos HTML <mark>:

    • <mark>: El elemento HTML <mark> define el texto que debe marcarse o resaltarse.
     <p> No olvides comprar <mark> leche </mark> hoy.</p>

    En este ejemplo, el texto "leche" aparece resaltado en el contexto de una frase.

    Elementos HTML <del>:

    • <del>: El elemento HTML <del> define el texto que se ha eliminado de un documento.
     <p> Mi color favorito es <del>azul</del> rojo.</p>

    En este ejemplo, el texto "azul" aparece tachado, indicando que ha sido eliminado y reemplazado por "rojo".

    Elementos HTML <ins>:

    • <ins>: El elemento HTML <ins> define el texto que se ha insertado en un documento.
     <p>Mi color favorito es <del> azul</del> <ins> rojo</ins>.</p>

    En este ejemplo, el texto "rojo" aparece subrayado, indicando que es texto recién insertado en lugar de "azul".

    Elementos HTML <sub>:

    • <sub>: El elemento HTML <sub> define el texto en subíndice.
     <p>Este es un texto <sub>subscrito</sub> text.</p>

    En este ejemplo, el texto "subscrito" aparece en subíndice, útil para fórmulas como H2O.

    Elementos HTML <sup>:

    • <sup>: El elemento HTML <sup> define el texto en superíndice.
     <p>Este es un texto  <sup>superindice</sup> text.</p>

    En este ejemplo, el texto "super indice" aparece en superíndice, útil para notas al pie o referencias como WWW[1].
