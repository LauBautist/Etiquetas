¿Qué son los colores en HTML?

En HTML, puedes aplicar colores a textos, fondos y otros elementos utilizando CSS. Los colores pueden ser definidos de diversas formas: utilizando nombres de colores, valores hexadecimales, valores RGB o HSL.

Formas de Definir Colores

1. Nombres de Colores

HTML tiene una lista de nombres de colores predefinidos como red, blue o green.

Ejemplo:

            color: red;

2. Valores Hexadecimales

Los colores también pueden ser definidos con valores hexadecimales. Estos comienzan con el símbolo # seguido de seis dígitos que representan los niveles de rojo, verde y azul.

Ejemplo:

            color: #ff0000;  /* Rojo */

3. Valores RGB

Los valores RGB utilizan números que indican la intensidad de rojo, verde y azul en el color. Los valores están en el rango de 0 a 255.

Ejemplo:

            color: rgb(255, 0, 0);  /* Rojo */

4. Valores HSL

La notación HSL define un color en términos de su matiz (hue), saturación y luminosidad. Los valores de matiz van de 0 a 360, y la saturación y luminosidad se expresan en porcentaje.

Ejemplo:

            color: hsl(0, 100%, 50%);  /* Rojo */

Colores de Bordes y Fondos

Además de aplicar colores al texto, puedes definir colores para los *bordes* y el *fondo* de los elementos HTML utilizando las propiedades border-color y background-color en CSS.

1. Color de Fondo

El color de fondo de un elemento se define con background-color.

Ejemplo:

            background-color: #f2e9fa;  /* Color lila claro */

2. Color de Borde

El color del borde de un elemento se define con border-color. Es necesario definir primero un estilo de borde.

Ejemplo:

            border: 2px solid rgb(141, 76, 167);  /* Borde morado */
