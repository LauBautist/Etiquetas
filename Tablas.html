Las tablas HTML permiten a los desarrolladores web organizar datos en filas y columnas.

Compañía Contacto País
Alfreds Futterkiste Maria Anders Alemania
Centro comercial Moctezuma Francisco Chang México
Ernst Handel Roland Mendel Austria
Island Trading Helen Bennett Reino Unido

Ejemplo:

Una tabla HTML sencilla:

        <table>
          <tr>
            <th>Compañía</th>
            <th>Contacto</th>
            <th>País</th>
          </tr>
          <tr>
            <td>Alfreds Futterkiste</td>
            <td>Maria Anders</td>
            <td>Alemania</td>
          </tr>
          <tr>
            <td>Centro comercial Moctezuma</td>
            <td>Francisco Chang</td>
            <td>México</td>
          </tr>
        </table>

Celdas de tabla

Cada celda de la tabla está definida por una etiqueta <td> y una </td>.

td significa datos de la tabla.

Todo lo que está entre <td> y </td> es el contenido de la celda de la tabla.

Ejemplo

    <table>
    <tr>
        <td>Emil</td>
        <td>Tobias</td>
        <td>Linus</td>
    </tr>
    </table>

Filas de la celda

Las filas de la tabla comienzan con una etiqueta <tr> y terminan con una etiqueta </tr>.

tr significa "fila de tabla".

    <table>
      <tr>
        <td>Emilio</td>
        <td>Tobias</td>
        <td>Linus</td>
      </tr>
      <tr>
        <td>16</td>
        <td>14</td>
        <td>10</td>
      </tr>
    </table>

Puedes tener tantas filas como desees en una tabla; sólo asegúrate de que el número de celdas sea el mismo en cada fila.

Encabezados de tabla

A veces, se desea que las celdas sean celdas de encabezado de tabla. En esos casos, utilice la etiqueta <th> en lugar de la etiqueta <td>: 


    <table>
      <tr>
        <th>Person 1</th>
        <th>Person 2</th>
        <th>Person 3</th>
      </tr>
      <tr>
        <td>Emil</td>
        <td>Tobias</td>
        <td>Linus</td>
      </tr>
      <tr>
        <td>16</td>
        <td>14</td>
        <td>10</td>
      </tr>
    </table>

Etiquetas de tabla HTML

Tag Description
<table> Define una tabla
<th> Define una celda de encabezado en una tabla
<tr> Define una fila en una tabla
<td> Define una celda en una tabla
<caption> Define un título para la tabla
<colgroup> Especifica un grupo de una o más columnas en una tabla para formato
<col> Especifica propiedades de las columnas dentro del elemento <colgroup>
<thead> Agrupa el contenido de encabezado en una tabla
<tbody> Agrupa el contenido del cuerpo en una tabla
<tfoot> Agrupa el contenido del pie en una tabla
