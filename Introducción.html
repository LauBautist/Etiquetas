¿Qué es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es un lenguaje de programación que define la estructura y el significado de las páginas web. Se trata de un código que indica al navegador cómo ordenar el contenido de una página web, como los textos, títulos, imágenes, videos y párrafos. El HTML es el componente básico de la web y se utiliza para crear páginas estáticas con elementos como tablas, listas, imágenes, encabezados y enlaces. El origen del HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN, propuso un sistema de hipertexto para compartir documentos. La idea era que los científicos pudieran compartir sus investigaciones para que sus colegas pudieran consultarlas fácilmente. Para crear un sitio web profesional, el HTML se complementa con otras tecnologías como las Hojas de Estilo en Cascada (CSS) y JavaScript. El CSS se encarga de los estilos de la página, como el fondo, los colores, el diseño, el espaciado y las animaciones. JavaScript añade funcionalidades dinámicas como galerías de fotos, sliders y pop-ups.

Estructura básica: 

    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Página HTML Simple</title>
    </head>

    <body>
        <h1>¡Bienvenido a mi página web!</h1>
        <p>Este es un ejemplo de un documento HTML básico.</p>
        <a href="https://www.ejemplo.com">Haz clic aquí para ir a Ejemplo.com</a>

    </body>
    </html>

Ejemplo explicado

  • La <!DOCTYPE html> : define que este documento es un documento HTML5.
  • El <html lang="es">: es el elemento raíz de una página HTML.
  • El <head>: contiene metainformación sobre la página HTML.
  • El <meta charset="UTF-8">: define la codificación de caracteres del documento.
  • El <meta name="viewport" content="width=device-width, initial-scale=1.0">: configura la vista para dispositivos móviles y de escritorio.
  • El <title>Página HTML Simple</title>: define el título del documento.
  • El <body>: define el cuerpo del documento y contiene todo el contenido visible.
  • El <h1>¡Bienvenido a mi página web!</h1>: es un encabezado principal de nivel 1.
  • El <p>Este es un ejemplo de un documento HTML básico.</p>: define un párrafo de texto.
  • El <a href="https://www.ejemplo.com">Haz clic aquí para ir a Ejemplo.com</a>: es un enlace que lleva al usuario a una URL externa.

Historial de HTML

Desde los primeros días de la World Wide Web, ha habido muchas versiones de HTML:

Año Versión
1989 Tim Berners-Lee inventó la World Wide Web (www)
1991 Tim Berners-Lee inventó HTML
1993 Dave Raggett redactó HTML+
1995 El Grupo de Trabajo de HTML definió HTML 2.0
1997 Recomendación del W3C: HTML 3.2
1999 Recomendación del W3C: HTML 4.01
2000 Recomendación del W3C: XHTML 1.0
2008 Primer Borrador Público del WHATWG: HTML5
2012 Estándar Viviente del WHATWG: HTML5
2014 Recomendación del W3C: HTML5
2016 Recomendación Candidata del W3C: HTML 5.1
2017 Recomendación del W3C: HTML5.1 (2ª Edición)
2017 Recomendación del W3C: HTML5.2
