Definicion

Los enlaces HTML, también conocidos como hipervínculos, son elementos de una página web que permiten a los usuarios saltar a otro contenido al hacer clic en ellos. Los enlaces pueden ser texto, imágenes u otros elementos HTML. Para insertar un enlace en HTML se utiliza el comando < A > (de anchor), que requiere de ciertos parámetros para funcionar. Uno de estos parámetros es HREF, que indica hacia dónde debe ir el enlace.

Enlaces HTML - Sintaxis

La etiqueta HTML <a> define un hipervínculo. Tiene la siguiente sintaxis:

 <a href="url">Texto del enlace</a>

El atributo más importante del elemento <a> es el atributo href, que indica el destino del enlace. El texto del enlace es la parte que será visible para el lector. Al hacer clic en él, el usuario será enviado a la URL especificada.

Ejemplo

 <a href="https://www.w3schools.com/">Visita W3Schools.com!</a>

De forma predeterminada, los enlaces aparecen de la siguiente manera en los navegadores:

  • Un enlace no visitado está subrayado y en azul.
  • Un enlace visitado está subrayado y en color morado.
  • Un enlace activo está subrayado y en rojo.

Enlaces HTML - El atributo target

Por defecto, la página vinculada se mostrará en la ventana actual del navegador. Para cambiar esto, se utiliza el atributo target, que especifica dónde abrir el documento vinculado.

  • _self - Predeterminado. Abre el documento en la misma ventana o pestaña.
  • _blank - Abre el documento en una nueva ventana o pestaña.
  • _parent - Abre el documento en el marco principal.
  • _top - Abre el documento en el cuerpo completo de la ventana.

Ejemplo

 <a href="https://www.w3schools.com/" target="_blank">Visita W3Schools!</a>

URL absolutas vs. URL relativas

Se pueden utilizar URLs absolutas o relativas en el atributo href.

Ejemplo de URLs absolutas

 <a href="https://www.w3.org/">W3C</a>
 <a href="https://www.google.com/">Google</a>

Ejemplo de URLs relativas

 <a href="html_images.asp">Imágenes HTML</a>
 <a href="/css/default.asp">Tutorial de CSS</a>

Enlaces HTML: usar una imagen como enlace

Para usar una imagen como enlace, coloca la etiqueta <img> dentro de la etiqueta <a>:

Ejemplo

 <a href="default.asp">
 <img src="smiley.gif" alt="Tutorial HTML" style="width:42px;height:42px;">
 </a>

Enlace a una dirección de correo electrónico

Utiliza mailto: en el atributo href para crear un enlace que abra el programa de correo electrónico del usuario:

Ejemplo

 <a href="mailto:someone@example.com">Enviar correo</a>

Botón como enlace

Para utilizar un botón HTML como enlace, es necesario agregar código JavaScript.

Ejemplo

 <button onclick="document.location='default.asp'">Tutorial HTML</button>

Títulos de enlaces

El atributo title especifica información adicional sobre un elemento y se muestra como un texto emergente cuando el ratón pasa sobre el enlace.

Ejemplo

 <a href="https://www.w3schools.com/html/" title="Ir a la sección HTML de W3Schools">Visita nuestro Tutorial de HTML</a>
