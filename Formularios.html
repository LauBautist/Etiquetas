Formulario Básico

Los formularios en HTML permiten a los usuarios ingresar datos que pueden ser enviados al servidor para su procesamiento. Aquí tienes un ejemplo básico de un formulario:

    <form action="/submit" method="POST">
        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre" required>

        <label for="correo">Correo Electrónico:</label>
        <input type="email" id="correo" name="correo" required>

        <label for="mensaje">Mensaje:</label>
        <textarea id="mensaje" name="mensaje" required></textarea>

        <button type="submit">Enviar</button>
    </form>

Atributos de Formularios

Un formulario en HTML tiene varios atributos importantes que determinan cómo se envían y procesan los datos:

  • action: Especifica la URL a la que se enviarán los datos del formulario.
  • method: Define el método HTTP utilizado para enviar los datos (GET o POST).
  • enctype: Especifica cómo se codifican los datos cuando se envían al servidor (por ejemplo, multipart/form-data).

Ejemplo de un formulario con atributos:

    <form action="/procesar" method="GET">
        <label for="busqueda">Buscar:</label>
        <input type="text" id="busqueda" name="q">

        <button type="submit">Buscar</button>
    </form>

Elementos de Formularios

Dentro de un formulario, puedes utilizar varios elementos para recopilar diferentes tipos de datos del usuario. Estos son algunos de los elementos más comunes:

  • <input>: Campo de entrada para texto, correo, contraseñas, etc.
  • <textarea>: Campo para escribir textos más largos.
  • <select>: Menú desplegable de opciones.
  • <button>: Botón para enviar el formulario.

Ejemplo de uso de varios elementos:

    <form>
        <label for="pais">País:</label>
        <select id="pais" name="pais">
            <option value="mx">México</option>
            <option value="es">España</option>
            <option value="ar">Argentina</option>
        </select>

        <label for="comentarios">Comentarios:</label>
        <textarea id="comentarios" name="comentarios"></textarea>

        <button type="submit">Enviar</button>
    </form>

Tipos de Entrada en HTML

Los diferentes tipos de entradas que puedes utilizar en un formulario de HTML se definen a través del atributo type en el elemento <input>. Algunos ejemplos son:

  • text: Entrada de texto simple.
  • email: Entrada para direcciones de correo electrónico.
  • password: Entrada de contraseñas (los caracteres se ocultan).
  • number: Entrada para números.
  • date: Selector de fecha.

Ejemplo de varios tipos de entradas:

    <form>
        <label for="edad">Edad:</label>
        <input type="number" id="edad" name="edad" min="18" max="100">

        <label for="fecha">Fecha de Nacimiento:</label>
        <input type="date" id="fecha" name="fecha">

        <label for="correo2">Correo Electrónico:</label>
        <input type="email" id="correo2" name="correo2">

        <button type="submit">Enviar</button>
    </form>

Atributos de Entrada

Los campos de entrada en un formulario pueden tener varios atributos que afectan su comportamiento. Algunos atributos importantes son:

  • required: Obliga al usuario a completar el campo antes de enviar el formulario.
  • placeholder: Muestra un texto temporal dentro del campo que desaparece al escribir.
  • min y max: Establecen los valores mínimo y máximo para tipos de entrada como number y date.
  • pattern: Especifica una expresión regular que el valor del campo debe cumplir.

Ejemplo de atributos en campos de entrada:

    <form>
        <label for="telefono">Teléfono (formato 10 dígitos):</label>
        <input type="tel" id="telefono" name="telefono" pattern="[0-9]{10}" required>

        <label for="password">Contraseña (mínimo 8 caracteres):</label>
        <input type="password" id="password" name="password" minlength="8" required>

        <button type="submit">Enviar</button>
    </form>
