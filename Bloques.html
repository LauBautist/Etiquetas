Bloques HTML y Elementos en Línea

En HTML, los elementos se dividen en dos grandes categorías: *elementos de bloque* y *elementos en línea*. Esta diferenciación es importante para entender cómo se comportan los elementos dentro de una página web.

Elementos de Bloque

Los *elementos de bloque* son aquellos que ocupan todo el ancho disponible de su contenedor. Esto significa que siempre comienzan en una nueva línea y pueden contener otros elementos de bloque o elementos en línea. Los elementos de bloque se utilizan para estructurar el contenido en bloques grandes y visualmente separados.

  • Empiezan en una nueva línea y ocupan el ancho completo del contenedor.
  • Pueden contener otros elementos de bloque o elementos en línea.
  • Utilizados para agrupar contenido o crear secciones de la página.

Elementos en Línea

Los *elementos en línea* solo ocupan el espacio necesario para su contenido. No inician en una nueva línea, lo que permite que múltiples elementos en línea puedan aparecer uno al lado del otro. Son ideales para dar estilo o agrupar pequeñas partes de texto.

  • No inician en una nueva línea.
  • Solo ocupan el espacio necesario para su contenido.
  • No pueden contener elementos de bloque, pero sí otros elementos en línea.

Ejemplo de Elementos en Bloque y en Línea:

Este es un elemento de bloque, usando la etiqueta <div>.

Este es un párrafo, que también es un elemento de bloque.

Este es un texto en línea dentro de un párrafo, que es un elemento de bloque.

<div>
    Este es un elemento de bloque usando la etiqueta <div>.
</div>

<p>
    Este es un párrafo con texto en línea como <span> dentro de él.
</p>

El elemento <div>

El elemento <div> es uno de los más utilizados en HTML para agrupar contenido. No tiene ningún significado semántico, pero es extremadamente útil para organizar y dar estilo a las secciones de una página web. Se comporta como un elemento de bloque.

Título dentro de un div

Este es un párrafo dentro de un <div>. Puedes agrupar varios elementos dentro de un <div> para aplicarles un estilo o manipularlos de forma conjunta. 

<div>
    <h3>Título dentro de un div</h3>
    <p>Este es un párrafo dentro de un <div>...</p>
</div>

El elemento <span>

El elemento <span> es un contenedor genérico de tipo en línea. Se usa principalmente para aplicar estilos a una porción específica de texto o contenido dentro de otro elemento. A diferencia de <div>, <span> no crea un nuevo bloque, sino que permanece en línea con el contenido circundante.

Ejemplo de uso de <span>:

Este es un párrafo en el que un texto en línea está resaltado utilizando <span>. 

<p>
    Este es un párrafo con un <span> usado para resaltar una porción del texto.
</p>
