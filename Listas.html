Listas Desordenadas (<ul>)

Las listas desordenadas se utilizan cuando el orden de los elementos no importa. Los elementos de la lista están marcados con viñetas.

Ejemplo de Lista Desordenada

  • Elemento 1
  • Elemento 2
  • Elemento 3

Para crear una lista desordenada, se usa la etiqueta <ul> para la lista y la etiqueta <li> para cada elemento de la lista. 

    <ul>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ul>

Listas Ordenadas (<ol>)

Las listas ordenadas se utilizan cuando el orden de los elementos es importante. Cada elemento de la lista está numerado.

Ejemplo de Lista Ordenada

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Para crear una lista ordenada, se usa la etiqueta <ol> para la lista y la etiqueta <li> para cada elemento de la lista. 

    <ol>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ol>

Listas de Definiciones (<dl>, <dt>, <dd>)

Las listas de definiciones se utilizan para listar términos y sus definiciones correspondientes. Se usa la etiqueta <dl> para la lista de definiciones, <dt> para definir el término y <dd> para la descripción o definición.

Ejemplo de Lista de Definiciones

HTML
Lenguaje de marcado para crear páginas web.
CSS
Lenguaje de estilos utilizado para dar diseño a las páginas web.
JavaScript
Lenguaje de programación utilizado para agregar interactividad a las páginas web.

Para crear una lista de definiciones, se usa la etiqueta <dl> para la lista general, <dt> para los términos y <dd> para las definiciones. 

    <dl>
        <dt>HTML</dt>
        <dd>Lenguaje de marcado para crear páginas web.</dd>

        <dt>CSS</dt>
        <dd>Lenguaje de estilos utilizado para dar diseño a las páginas web.</dd>

        <dt>JavaScript</dt>
        <dd>Lenguaje de programación utilizado para agregar interactividad a las páginas web.</dd>
    </dl>

Etiquetas de Lista en HTML

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define un elemento de lista
<dl> Define una lista de descripciones
<dt> Define un término en una lista de descripciones
<dd> Describe el término en una lista de descripciones
