Imagenes de HTML

Las imágenes pueden mejorar el diseño y la apariencia de una página web

Ejemplos

A continuación se muestra un ejemplo básico de cómo utilizar la etiqueta <img> en HTML: 

 <img src="pic_trulli.jpg" alt="Trulli Italiano">

En este caso, src define la ubicación del archivo de la imagen, y alt proporciona un texto alternativo que se mostrará si la imagen no puede cargarse.

 <img src="img_girl.jpg" alt="Chica con chaqueta">

En este ejemplo, src indica la ubicación de la imagen, mientras que alt es el texto alternativo que describe la imagen como "Chica con chaqueta" si no puede cargarse.

 <img src="img_chania.jpg" alt="Flores en Chania">

En este ejemplo, src indica la ubicación de la imagen, mientras que alt es el texto alternativo que describe la imagen como "Flores en Chania" si no puede cargarse.

Sintaxis de imágenes HTML

La <img> etiqueta HTML se utiliza para incrustar una imagen en una página web.

Técnicamente, las imágenes no se insertan en una página web, sino que se vinculan a páginas web. La <img> etiqueta crea un espacio de almacenamiento para la imagen a la que se hace referencia.

La <img> etiqueta está vacía, contiene solo atributos y no tiene una etiqueta de cierre.

Atributos obligatorios de la etiqueta <img>:

  1. src - Especifica la ruta a la imagen.
  2. alt - Especifica un texto alternativo para la imagen.
 <img src="url" alt="alternatetext">

En este ejemplo, src es el atributo que especifica la URL de la imagen, y alt es el texto alternativo que se mostrará si la imagen no se puede cargar.

El atributo src

El atributo requerido src especifica la ruta (URL) a la imagen.

Nota: Cuando se carga una página web, es el navegador el que, en ese momento, obtiene la imagen de un servidor web y la inserta en la página. Por lo tanto, asegúrese de que la imagen permanezca en el mismo lugar en relación con la página web; de lo contrario, sus visitantes verán un icono de enlace roto. El icono de enlace roto y el alt texto se muestran si el navegador no puede encontrar la imagen.

Atributo alt

El atributo alt proporciona un texto alternativo para una imagen en caso de que no pueda verse por alguna razón (conexión lenta, error en el atributo src o si el usuario utiliza un lector de pantalla).

El valor del atributo alt debe describir la imagen:

 <img src="img_chania.jpg" alt="Flores en Chania">

Si el navegador no encuentra una imagen, mostrará el valor del atributo alt:

 <img src="wrongname.gif" alt="Flores en Chania">

Tamaño de la imagen: ancho y alto

Puedes utilizar el atributo style para especificar el ancho y la altura de una imagen:

 <img src="img_paisaje.jpg" alt="Paisaje" style="width:150px;height:150px;">

Alternativamente, puedes usar los atributos width y height:

 <img src="img_paisaje.jpg" alt="paisaje" width="150" height="150">

Es importante especificar el ancho y la altura de las imágenes para evitar que la página parpadee mientras las imágenes se cargan.

Imágenes en otra carpeta

Si tienes tus imágenes en una subcarpeta, debes incluir el nombre de la carpeta en el atributo src: 

 <img src="/imagenes/html5.gif" alt="Icono de HTML5" style="width:128px;height:128px;">

Imágenes en otro servidor o sitio web

Para apuntar a una imagen en otro servidor, necesitas especificar una URL completa en el atributo src: 

 <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Las imágenes externas pueden estar sujetas a derechos de autor. Asegúrate de obtener permiso antes de usarlas.

Imágenes animadas

HTML permite el uso de GIF animados:

 <img src="programming.gif" alt="Hombre programando" style="width:48px;height:48px;">

Imagen como enlace

Para usar una imagen como enlace, coloca la etiqueta <img> dentro de la etiqueta <a>: 

 <a href="default.asp">
 <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
 </a>

Imagen flotante

Utiliza la propiedad float en CSS para hacer que una imagen flote a la derecha o a la izquierda del texto:

 <p><img src="smiley.gif" alt="Carita feliz" style="float:right;width:42px;height:42px;">
 El texto se ajusta a la izquierda de la imagen.</p> 
 <p><img src="smiley.gif" alt="Carita feliz" style="float:left;width:42px;height:42px;">
 El texto se ajusta a la derecha de la imagen.</p>
